# 🤖 AI Chatbot (Groq + Streamlit)

A simple but powerful AI chatbot built using **Streamlit** and **Groq API**.  
It uses the **GPT OSS 120B** to generate fast, intelligent, and human-like responses.

---

## 🚀 Features

- 💬 Real-time AI chatbot
- ⚡ Fast responses using Groq API
- 🌙 Dark modern UI
- 🧠 Uses GPT OSS 120B model
- 📝 Markdown formatted AI responses
- 🔐 Secure API key handling with `.env`

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🎈
- Groq API ⚡
- OpenAI SDK (compatible client)
